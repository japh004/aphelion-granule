A modern, comprehensive platform connecting driving students with top-rated driving schools in Cameroon.

## Overview

Drissman simplifies the process of getting a driver's license by providing a seamless digital experience for both students and driving schools. Students can search, compare, and book lessons, while schools get powerful tools to manage their business.

## üöÄ Key Features

### For Students
- **Smart Search**: Find driving schools by location, price, and success rates.
- **Rich Profiles**: detailed school information, photo galleries, and verified reviews.
- **Online Booking**: Securely book driving lessons and code courses.
- **Dashboard**: Track training progress, upcoming sessions, and payments.
- **Code de la Route**: Integrated curriculum and interactive practice quizzes.

### For Driving Schools (Partners)
- **Digital Presence**: Professional profile page to attract more students.
- **Management Dashboard**: Overview of bookings, revenue, and student progress.
- **Fleet & Instructor Management**: (Upcoming) Tools to organize resources.

## üèÅ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/drissman-frontend.git
   cd drissman-frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser.

## üìÅ Project Structure

- `src/app`: Application routes and page components.
- `src/components`: Reusable UI components (Atomic design).
  - `/ui`: Generic base components (Button, Input, etc.).
  - `/shared`: Cross-functional components (NavBar, Footer).
  - `/features`: Specific feature components (Search, Auth, etc.).
- `src/lib`:/utils`: Helper functions and constants.
- `public`: Static assets (images, fonts).
